# 🛒 E-Commerce Store (HTML, CSS & JavaScript)

## 📌 Project Description
This project is a **front-end E-commerce Store web application** built using **HTML, CSS, and JavaScript only**. It demonstrates how an online shopping website works by allowing users to browse products, view details, and manage a shopping cart dynamically.

The main goal of this project is to practice **front-end web development**, improve UI design skills, and gain hands-on experience with **JavaScript DOM manipulation and event handling**. All functionality is handled on the client side without using any backend or database.

---

## ✨ Features
- Product listing with images, names, and prices  
- Dynamic **Add to Cart** functionality  
- Update and remove items from cart  
- Real-time cart total calculation  
- Responsive and user-friendly design  
- Interactive UI using JavaScript  

---

## 🛠️ Technologies Used
- **HTML** – Page structure and content  
- **CSS** – Styling and responsive layout  
- **JavaScript** – Interactivity and cart logic  

---

---

## 🚀 How to Run the Project
1. Download or clone this repository  
2. Open the `index.html` file in any modern web browser  
3. Browse products and use the cart functionality

- Improved UI/UX design skills  
- Experience building a real-world web application
